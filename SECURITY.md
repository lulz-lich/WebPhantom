# Security Policy

## Supported Versions

The latest release is the supported version for security fixes.

## Reporting a Security Issue

Please do not publish security-sensitive reports as public issues. Open a private advisory on GitHub or contact the maintainer through the repository profile.

Include:

- Affected version or commit.
- Reproduction steps.
- Impact.
- Whether sensitive data may be exposed.

## Project Safety Boundary

WebPhantom is passive browser automation for authorized assessment documentation. Reports requesting malware behavior, credential theft, exploitation, stealth, persistence, or unauthorized access are out of scope.
